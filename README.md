# 💕 Website Pribadi Karisa Fadlina

Website pribadi yang dibuat untuk Karisa Fadlina dengan penuh cinta oleh Rifqi Saefulloh.

## 🌸 Fitur Website

- **Halaman Pribadi** - Hanya bisa diakses dengan kode akses
- **Countdown** - Menghitung hari menuju tanggal 9 setiap bulan
- **Perayaan** - Otomatis merayakan setiap tanggal 9
- **Surat Cinta** - Pesan romantis dari Rifqi
- **Galeri** - Momen-momen special bersama

## 🔐 Kode Akses

```
rifqikarisaku
```

## 💍 Wedding Day

Setiap tanggal 9 adalah hari pernikahan Rifqi & Karini - website akan otomatis merayakan!

## 🚀 Deploy ke GitHub Pages

1. Buat repository baru di GitHub
2. Push semua file ke repository
3. Aktifkan GitHub Pages di Settings > Pages
4. Pilih "GitHub Actions" sebagai source

## 📝 Struktur File

```
karisa-fadlina/
├── index.html    # Halaman utama
├── style.css     # Styling
├── script.js     # JavaScript
└── README.md     # File ini
```
